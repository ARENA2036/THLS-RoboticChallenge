## Robotik Challenge 2026 

**Teil des Transformations-Hub Leitungssatz** an der 
**ARENA2036** – Forschungscampus Stuttgart
---

Der [Transformations-Hub Leitungssatz](https://www.leitungssatz-hub.de) veranstaltet die [Robotik Challenge](https://www.leitungssatz-hub.de/robotik-challenge/robotik-challenge-2026/), um die Automatisierung komplexer manueller Prozesse in der Leitungssatzmontage voranzutreiben. Eine [Begleitforschung untersucht Digitalisierungsaspekte](https://www.leitungssatz-hub.de/robotik-challenge/robotik-challenge-2026/begleitforschungs-und-digitalisierungsmodul/), deren Inhalte werden vorwiegend in diesem Repository veröffentlicht.

Dieses Repository dient der transparenten Bereitstellung der Challenge-Daten und fördert den offenen Innovationsaustausch.
